信奥赛智能学习诊断系统

为 CSP-J/S 信奥赛学生提供水平诊断、个性化路径规划、每日题目推荐、错题AI归因， 并生成家长能看懂的进度报告 —— 解决"学生盲目刷题、家长无法辅导、老师时间不够"的三方痛点

🎯 项目背景

痛点场景

作为被C++折磨了三年的信奥赛家长，孩子学到动态规划那晚，拿来一道WA的题问我"状态转移方程怎么写"， 我盯着 dp[i][j] = max(...) 脑子一片空白。那一刻我意识到：我辅导不了他了。 市面上的工具要么只有题库（洛谷），要么不懂信奥体系（通用AI），没人解决"诊断+规划+归因"这一段空白。

本系统专为 CSP-J/S 信奥赛备考学生设计，通过 AI 技术实现：
• 精准诊断：评估学生当前水平，找出薄弱知识点
• 路径规划：生成个性化学习路线，避免盲目刷题
• 智能推荐：每日推荐针对性练习题目
• 错题归因：AI 分析错误原因，提供改进建议
• 家长报告：生成易懂的进度报告，让家长了解学习情况

✨ 核心功能

📊

诊断中心

支持 CSP-J（入门组）和 CSP-S（提高组）两级诊断测试， 通过多维度题目评估，生成详细的能力雷达图和薄弱点分析报告

🗺️

学习路径

基于诊断结果生成个性化学习路线图，知识点按依赖关系解锁， 实时显示掌握进度，引导学生循序渐进学习

📝

每日推荐

根据薄弱知识点智能推荐每日练习题目，内置 Monaco 代码编辑器， 支持在线编写 C++ 代码并模拟运行测试

🔍

错题归因

收集所有错题并使用 AI 进行深度分析，识别错误类型（答案错误/运行时错误等）， 提供根本原因分析和改进建议

📈

家长报告

生成可视化学习进度报告，包含近7天学习情况、知识点掌握度图表， 用通俗易懂的语言帮助家长了解孩子学习状态

💾

数据持久化

使用 localStorage 本地存储用户数据，无需后端服务器， 支持学习进度、错题记录、诊断历史的完整保存

💡 功能演示

诊断测试界面

选择 CSP-J 或 CSP-S 级别后进入诊断测试，系统根据知识点分布生成测试题目， 完成后自动生成诊断报告，包含各知识点得分和整体能力评估

学习路径可视化

以时间线形式展示学习路径，每个节点代表一个知识点，显示当前掌握度百分比， 未解锁节点显示前置依赖条件，引导学生有序学习

代码编辑器

集成 Monaco Editor（VS Code 同款编辑器），支持 C++ 语法高亮、代码补全， 提供一键运行测试功能，实时显示测试结果和错误信息

AI 错题分析

自动识别错误类型（答案错误、运行时错误、超时等），结合知识点给出根本原因分析， 提供代码对比和改进建议，帮助学生理解错误本质

🛠️ 技术栈

React 18 TypeScript Vite 5 Tailwind CSS ECharts Monaco Editor Framer Motion Zustand React Router v6 localStorage

架构特点：纯前端架构，无需后端服务器，所有数据存储在本地。 使用 TRAE AI 辅助开发，从需求分析、架构设计到代码实现全程由 AI 协助完成。
